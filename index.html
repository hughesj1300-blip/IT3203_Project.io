Introduction to Web Browsers

Web browsers are a fundamental and ubiquitous component of today's web. They are the applications which provide users with an interface to access, navigate and view every website and webpage available on the internet [1]. The web browser acts as the client in the client-server architecture and is responsible for correctly retrieving and displaying webpages for the user.
